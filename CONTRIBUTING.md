# Contributing to AfterMe

First off, thank you for your interest in contributing.  
Second, understand the following clearly:

This is a **proprietary project** owned by **Tyler Hammond**.  
All contributions are made under the agreement that:

- You assign all rights to the project owner.  
- You do not retain ownership of submitted code, ideas, or designs.  
- You may not reuse or republish contributions elsewhere.

## How to Contribute

1. Obtain written permission from Tyler Hammond before contributing.
2. Create a pull request that:
   - Describes the change
   - References any related tasks
   - Includes testing notes if applicable
3. All code must follow project structure and conventions.
4. Contributions may be modified, rejected, or removed at any time.
